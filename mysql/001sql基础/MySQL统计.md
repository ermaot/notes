AVG(score) AS mean,STDDEV_SAMP(score) AS 'gtd. dev.' ,VAR_SAMP( score) AS 'variance FROM testscore;

## 分组

```
SELECT COUNT (score) AS n,SUM(score) AS sum,MIN(score) AS minimum,MAX( score) AS maximum,
AVG(score) AS mean,STDDEV_SAMP(score) AS 'gtd. dev.' ,VAR_SAMP( score) AS 'variance FROM testscore group by sex;
```
